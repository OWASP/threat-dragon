## Security Policy

If you find a vulnerability anywhere in this project, such as the source or scripts,
then please let the project admins know ASAP and we will fix it as a critical priority.

Open a [security advisory][advisory] which will be provided
only to Threat Dragon's admins and in strict confidence.

We welcome analysis and proof of concepts generated by AI,
but for the advisory to be accepted it must have had human oversite and checking.

[advisory]: https://github.com/OWASP/threat-dragon/security/advisories/new
