Secure Storage Unavailable

Electron's safeStorage API has fallen back to basic_text encryption, which is not secure enough for storing sensitive data. For more information, see the Electron safeStorage documentation.
Your API key will not be saved on disk as plain text. Other settings (model, temperature, etc.) will be saved normally.

Running via Snap

To enable secure storage:

  1. Run the following command in a terminal:
    sudo snap connect threat-dragon-ng:password-manager-service
  2. Close and re-launch Threat Dragon

Linux with Unsupported Window Manager

If you're running on Linux with an unsupported window manager, you can launch Threat Dragon with a flag to specify the secret store:

threat-dragon --password-store=gnome-libsecret

Other supported values include: basic, kwallet, kwallet5, kwallet6
